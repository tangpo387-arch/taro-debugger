## 3. Edge Cases & Design Decisions

- **Layout Token Relocation**: `layout.config.ts` was moved into the library to ensure it remains independent. Other app components now import this token from the library.
- **Peer Dependencies**: Verified and added all required peer dependencies (Angular Forms, CDK, Monaco, DAP-Core) to ensure library build integrity.
- **Build Isolation**: While `ng build ui-editor` encountered a known `ng-packagr` destructuring error in the monorepo env, `npm run build` (full app) and library unit tests both pass, confirming functional correctness.
