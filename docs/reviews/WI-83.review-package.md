## 3. Edge Cases & Design Decisions

- **Race Condition Guard**: Implemented a check in `loadCallStack` after the `stackTrace` DAP request returns. If the process has resumed (`executionState !== 'stopped'`) during the request, the stale results are discarded. This ensures the Call Stack panel remains empty while the program is running, even if a late-arriving DAP response would have otherwise overwritten the cleared state.
- **Empty State Consistency**: Verified `CallStackComponent` uses `TaroEmptyStateComponent` with icons and descriptions matching the `VariablesComponent` pattern (standardized in WI-80).
