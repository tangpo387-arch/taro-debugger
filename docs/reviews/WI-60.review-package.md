## 3. Edge Cases & Design Decisions

- **DI Tokenization**: Decoupled `IpcTransportService` from the global `window` object by using `ELECTRON_API` token. This allows the library to be used in non-Electron environments (graceful failure) or with different IPC implementations.
- **Injector usage in Factory**: Changed `TransportFactoryService` to use `Injector` instead of `new` to ensure that transport instances (especially `IpcTransportService`) can access required DI tokens.
- **Shared Types**: `TransportType` was moved to `dap.types.ts` as it's a protocol-level configuration.
