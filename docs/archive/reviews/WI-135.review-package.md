## 3. Edge Cases & Design Decisions

* **Asset Loading Path Resolution:** No path manipulation was required inside `main.ts` because compiling workspace output to `dist/electron/main.js` maintains the exact relative layout of target Angular frontend assets in `dist/taro-debugger-frontend/browser/index.html` via `path.join(__dirname, '../taro-debugger-frontend/browser/index.html')`.
* **Redundant Script Removal:** Removed the `copy-assets.js` script since the Angular Application Builder compiles frontend files natively to `dist/` in standard builds.
