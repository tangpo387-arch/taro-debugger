## 3. Edge Cases & Design Decisions

- **Unified MatTooltip**: Replaced column-specific tooltips with a comprehensive `getInstructionTooltip` helper. This provides a consistent `<Address> <+Offset>: [<Opcode>] <Instruction>` format regardless of which cell is hovered, ensuring full context is always accessible even when columns are heavily truncated.
- **Smart Symbol Indentation**: To maintain column alignment (`ADDRESS:`, `OPCODE:`, `DISASM:`), extremely long C++ symbols are explicitly pushed to a new line with a 10-space indent (`          `). Strings exceeding 75 characters are chunked and indented to prevent the browser from wrapping text back to the left margin.
- **MatTooltip MDC Surface Styling**: The global stylesheet (`styles.scss`) was updated to target the `.mdc-tooltip__surface` class, enforcing `font-family: monospace`, `text-align: left`, and removing the "double-box" border effect caused by styling the host wrapper.
- **MatTooltip Implementation**: Standardized on `MatTooltip` with a `400ms` show delay. The previous native `[attr.title]` fallback was removed as it caused redundant tooltip rendering on modern browsers.
- **Flexbox Ellipsis Hardening**:
  - Set `min-width: 0` on flex items to override the default `min-width: auto` which can prevent shrinking.
  - Set `display: block` on flex items to ensure `text-overflow: ellipsis` is correctly interpreted by the rendering engine.
  - Added `max-width: 600px` to `.mnemonic` to prevent extremely long instructions from completely pushing the `.annotation` column off-screen.
- **High-Density Spacing**: Reduced the global `.assembly-row` gap to `8px` and adjusted column widths (`.address`: `112px`, `.offset`: `56px`) to ensure compact information display consistent with other debug panels.
- **Dynamic Offset Hiding**: The `.offset` column now uses `&:empty { display: none }` combined with a whitespace-trimmed template. This removes redundant gaps when offsets are not available, pulling the opcode closer to the address.
- **Viewport Clipping Fix**: The root cause of missing ellipsis was `.assembly-row` extending beyond `.assembly-viewport`. Added `max-width: 100%` and `overflow-x: hidden` to the viewport. This forces flex items to compress and trigger ellipsis.
