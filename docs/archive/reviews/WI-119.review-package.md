## 3. Edge Cases & Design Decisions

- **Shared Pipe Architecture**: Moving `CppSignaturePipe` to `ui-shared` avoids code duplication between the Call Stack (where it was already used) and the Assembly View.
- **Tooltip vs Label**: The display label is simplified for density, while the tooltip preserves the full mangled name for technical accuracy.
- **🔍 Inspect**: `projects/ui-shared/src/lib/pipes/cpp-signature.pipe.ts` — verify the nested bracket handling logic (`angleDepth` and `parenDepth`) covers complex template signatures without breaking.
