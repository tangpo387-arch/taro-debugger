## 3. Edge Cases & Design Decisions

- **Decoupled Selection**: Row clicks on thread nodes now only toggle expansion. This prevents accidental thread switches when navigating the tree.
- **Explicit Focus**: A dedicated `my_location` button appears on hover for non-active threads, providing a clear and intentional action for thread selection.
- **Auto-Expansion Persistence**: The `stopped` event reaction now overrides manual collapse for the active thread. This ensures that every time the debugger stops (or steps), the relevant stack is immediately visible.
- **🔍 Inspect**: `updateTree` in `thread-call-stack.component.ts` — verify that `restoreExpansion` and the new auto-expand logic correctly handle the tree state transition during rapid stepping.
