## 3. Edge Cases & Design Decisions

- **Sticky Tab Behavior**: The current logic already handles "stickiness" (if you are on Disassembly, you stay there). I ensured that clicking a frame without source while on Disassembly doesn't jump you back to Source.
- **Shared Component Enhancement**: I chose to enhance the shared `TaroEmptyStateComponent` with `<ng-content>` rather than creating a specialized component, ensuring consistency and reusability across the project.
- **Visual Clarity**: Description text in empty states is now allowed to wrap to multiple lines (up to 400px width) to prevent ellipsis truncation on longer informational messages.
