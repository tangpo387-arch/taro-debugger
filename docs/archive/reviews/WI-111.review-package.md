## 3. Edge Cases & Design Decisions

- **Three-Layer Alignment**: Successfully decoupled protocol-level caching (Session Layer) from viewport/UI stream management (UI Layer).
- **Fast-Path Optimization**: `DapAssemblyService.fetchInstructions` includes a fast-path that skips the cache call if the target address is already visible in the current UI sliding window.
- **Symbol Normalization**: Ported the logic to strip `<...>` and `+offset` from symbols to the core layer, ensuring consistent disassembly display regardless of UI implementation.
- **🔍 Inspect**: `DapAssemblyCacheService.fetchInstructions` (L84-118) — verify the gap-filling logic correctly calculates the `actualOffset` and `actualCount` when a partial cache hit occurs.
- **🔍 Inspect**: `DapAssemblyCacheService.pruneCache` (L235-256) — verify that spatial pruning correctly identifies the furthest ranges from the current IP.
