## 3. Edge Cases & Design Decisions

- **Base64 Conversion**: Used `atob` and `btoa` for Base64 conversion. These are available in both browser and Electron environments. For Node-based tests, Vitest handles these globally.
- **DAP Compliance**: `readMemory` and `writeMemory` calls include `this.ensureStopped()` in `DapSessionService` to enforce protocol-safe execution.
- **Reactive Updates**: `DapMemoryService.onMemoryUpdated$` allows UI components to refresh their view when a write operation occurs on a specific range.
- 🔍 Inspect: The `try...catch` blocks in `base64ToUint8Array` and `uint8ArrayToBase64` within `DapMemoryService.ts`. They provide graceful failure for malformed data from the adapter.
