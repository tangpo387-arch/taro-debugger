## 3. Edge Cases & Design Decisions

- **Decision: Removal of Virtual Scroll**: Replaced `AutoSizeVirtualScrollStrategy` with a plain scrollable div. Experimental CDK autosizing remains unstable for highly dynamic row heights (e.g., expanding massive JSON blocks). Native scroll ensures pixel-perfect bounds and 100% stability at the cost of slight rendering overhead for very long sessions (still well within performance targets for a diagnostic tool).
- **Decision: Payload Constraint**: Applied `max-height: 400px` to expanded JSON blocks with internal scrolling. This prevents a single massive "disassemble" response from breaking the UX and requires only moderate vertical scrolling to reach subsequent logs.
- **Micro-animation**: Maintained a smooth opacity transition for the view to hide layout shifts during complex tab transitions.
