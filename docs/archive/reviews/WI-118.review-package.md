## 3. Edge Cases & Design Decisions

- **Decision**: The "Jump to Address" button is now disabled while running (instead of hidden). This prevents flickering while respecting the DAP protocol's restriction on fetching disassembly in the running state (R6).
- **Decision**: Buttons in the floating action container are forced to `opacity: 1` when disabled, using background color shifts instead of transparency. This eliminates the "translucent flash" during rapid state transitions.
- **Decision**: The Instruction Pointer (arrow) is now dimmed (`.is-stale`) instead of removed during stepping. This provides visual continuity.
