## 3. Edge Cases & Design Decisions

- **Variables View Chevron click**: Resolves the double-toggle bug. By calling `$event.stopPropagation()` on the chevron button `(click)` handler, we prevent the event from bubbling up to the row element, which also has a toggle handler.
- **Thread Call Stack Focus Button**: Clicking the thread "Focus" button activates the thread. We call `$event.stopPropagation()` on the focus button `(click)` handler to prevent event bubbling from toggling the expansion of the thread node.
- **File Explorer View**: Natively uses `(click)="tree.toggle(node)"` on directories and has no other interactive buttons in the rows, so it fits the unified behavior perfectly without any code edits.
