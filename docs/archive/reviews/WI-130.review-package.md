## 3. Edge Cases & Design Decisions

- **Local Address State**: Component manages its `localBaseAddress` and dynamic `rows` locally. Any changes from parent inputs re-initialize the contiguous block, ensuring seamless initial jumps.
- **Scroll Anchoring Delta**: Sync offset changes are calculated as:
  $$\text{Offset}_{\text{New}} = \text{Offset}_{\text{Current}} + (\text{Rows}_{\text{Added}} \times \text{RowHeight})$$
- **16KB Memory Cap**: Memory viewport is capped at `1024` rows (`16KB` active buffer). Top rows are pruned when scrolling down and bottom rows are pruned when scrolling up to preserve small DOM footprints.
- 🔍 **Inspect**: The scroll anchoring implementation in `checkAndFetchMemory` during prepending.
