## 3. Edge Cases & Design Decisions

- **Super Root**: The `DapFileTreeService` now returns a synthetic "Super Root" (not visible in UI) to allow multiple visible root nodes (Project and External Libraries).
- **Sorting Exception**: Direct children of the "Super Root" are NOT sorted alphabetically to ensure the Project Root always appears before External Libraries.
- **Auto-Expansion**: Virtual roots are expanded only if `expandedPaths` is empty (initial load), avoiding overriding user state on dynamic reloads.
- 🔍 **Inspect**: `DapFileTreeService.buildTreeFromSources` logic for path categorization (isInsideRoot).
