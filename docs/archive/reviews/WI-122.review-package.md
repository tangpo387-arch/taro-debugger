## 3. Edge Cases & Design Decisions

- **Unified State**: Centralized the `isConnected` check in `DebuggerComponent.html` for main tabs (Source, Memory) to ensure a consistent full-panel empty state.
- **Component Self-Cleanup**: Components like `MemoryViewComponent` and `AssemblyViewComponent` now explicitly clear their internal data arrays on disconnect to prevent stale data from leaking if the tab is switched.
- **Icon Standardization**: Standardized on `link_off` for all "No active session" states across the IDE.
- **Font Synchronization**: Enforced `var(--font-sans)` within the `TaroEmptyStateComponent` to prevent inheritance of monospaced fonts when hosted in code-heavy views.
- **🔍 Inspect**: `DebuggerComponent.html` @if hierarchy. It now prioritizes the `!isConnected()` check over `activeFilePath` or `activeMemoryAddress` to ensure the empty state wins when the session dies.
