## 3. Edge Cases & Design Decisions

- **Visual Feedback**: With the removal of the spinner and buttons, visual feedback for "in-flight" requests is now handled via `cursor: wait` and reduced opacity on the input field (`.in-flight` class).
- **Error Handling**: Implemented `silentError` support in `DapSessionService.sendRequest`. The `evaluate` command now uses this flag to suppress the global `_dapError` event emission. This eliminates double-logging and prevents the global snackbar from appearing, while allowing `DebugConsoleComponent` to log the failure locally in a cleaner format (no "DAP Error" prefix).
- **Hotkey & Focus**: Implemented `Ctrl + Shift + Y` for console focus. The `KeyboardShortcutService` was updated to whitelist `evaluate-expression-input`, allowing debug shortcuts (F10, etc.) to remain active while typing in the console.
- **Styling Alignment**: Input text is now color-distinguished (`var(--mat-sys-primary)`) to separate active commands from log history.
