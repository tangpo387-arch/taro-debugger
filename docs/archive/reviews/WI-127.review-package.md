## 3. Edge Cases & Design Decisions

- Decision: Used a class-level `threadsQueryInFlight` Promise inside `DapSessionService.fetchThreads()` to throttle and coalesce parallel threads requests into a single in-flight request, satisfying AC-6.
- Decision: Implemented a constant `MIN_GAP_FETCH_SIZE = 50` to pre-cache instructions in bulk when traversing gaps, avoiding single-stepping high frequency queries.
- 🔍 Inspect: `getFromCache` inside `dap-assembly-cache.service.ts` — verify that the `Math.max(0, startIdx)` and `Math.min(range.instructions.length, ...)` slice logic bounds matches indices correctly and yields only valid subsets.
- 🔍 Inspect: `fetchInstructions` suffix/prefix calculation logic — verify that offset translation is correct and covers the missing range precisely.
