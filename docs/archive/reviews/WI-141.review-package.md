## 3. Edge Cases & Design Decisions

- **Dynamic Injector lookup**: Sub-managers resolve `DapSessionService` at runtime via `Injector.get(DapSessionService)` instead of standard constructor injection to break compile-time circular dependency loops.
- **Backward Compatibility**: Kept legacy getter properties (like `threadsSubject`, `breakpointsMap`) mapped as proxies in `DapSessionService` to avoid having to rewrite 2100 lines of existing integration test cases.
