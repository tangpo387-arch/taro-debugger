## 3. Edge Cases & Design Decisions

- Maintain `@internal Getters` for `breakpointsSubject`, `breakpointsMap`, `threadObjects`, `activeThreadSubject`, `threadsSubject`, and `systemBreakpointIds` to avoid breaking backward compatibility with existing tests and dependent UI components.
- The thread event processing employs a 50ms debouncer window (`flushThreadEventsBuffer`) in `DapThreadManager` to reduce redundant thread fetch overhead on high-frequency event streams, maintaining precise execution state tracking.
