## 3. Edge Cases & Design Decisions

> Explain any non-obvious decisions. Flag areas requiring deeper QCR inspection with 🔍.

- Decision: **Synthetic Events for Orchestration**. Instead of manually reloading the call stack inside `threads.component.ts`, `setCurrentThread()` emits a synthetic `stopped` event. This allows `DebuggerComponent` to maintain centralized control over stack trace fetching while keeping panels decoupled.
- Decision: **Elegant Stop Indicator**. Replaced the heavy textual badge with a minimal `pause_circle` icon and `matTooltip`. This preserves horizontal space in high-density panels while providing full stop-reason context on hover.
- 🔍 Inspect: Verify the state transition in `DapSessionService.handleTransportEvent` correctly clears `stoppedThreadId` upon `continued` or `terminated` events to avoid stale UI indicators.
