## 3. Edge Cases & Design Decisions

- **Naming Convention**: `InspectionPanelComponent` was renamed to `PanelComponent` (selector: `taro-panel`) to reflect its generic nature in the foundation library.
- **Dependency Flattening**: By moving `LAYOUT_COMPACT_MQ` to `ui-shared`, we successfully removed the dependency of `ui-inspection` and `ui-assembly` on `ui-editor` (Monaco).
- **Cascading Watch**: Designed the VS Code tasks to support "Binary-Linked" development. This ensures engineers always test against the compiled Angular Package Format, preventing `ngtsc` reference tracking bugs.
- 🔍 **Inspect**: `.vscode/tasks.json` — verify that the `problemMatcher` correctly detects both `Compilation complete` (library) and `bundle generation complete` (app) patterns.
