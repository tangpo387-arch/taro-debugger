## 3. Edge Cases & Design Decisions

> Explain any non-obvious decisions. Flag areas requiring deeper QCR inspection with 🔍.

- **Decision**: Added `this.executionStateSubject.next('running')` in the `catch` block of `terminate()`'s `sendRequest`.
- 🔍 Inspect: `terminate()` fallback logic (L249-L251) - When `terminate` request fails, the adapter didn't terminate correctly but we still want to disconnect.
