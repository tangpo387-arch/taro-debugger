# Visual Design Rules

This skill contains mapping for all UX/UI implementations, enforcing the project's adherence to the "Flush IDE" aesthetics and Dynamic Density System.

## 1. When to Use This Skill

You **MUST** load this skill before performing any of the following tasks:

- Writing or modifying SCSS files (`*.scss`)
- Adjusting HTML template margins, paddings, flex layouts, or grid
- Adjusting colors, font sizes, or typography weights
- Creating any new Angular Component that renders a visible element

## 2. Applicable Roles

- **Lead_Engineer**: Must read before writing or modifying any SCSS or templates.
- **Quality_Control_Reviewer**: Must read before reviewing the implementation of UI components to ensure exact adherence to the Flush IDE principles.
